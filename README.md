##Repertoire téléphonique

Version 1
Permet de rentrer plusieurs informations de la personne.


## Fonctionnalité

Permet de rentrer plusieurs informations de la personne :

    - Nom et Prénom
    - Téléphone
    - Pseudo
    - Adresse postale
    - Email
    - date de naissance
    - Note....

    Classe le contenu par ordre alphabetique.
